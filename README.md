
# Login & Signup Form

A modern, animated authentication form with dual login and signup panels featuring smooth transitions and responsive design.

## Features

- **Dual Panel Design**: Toggle between login and signup forms with smooth animations
- **Modern UI**: Gradient backgrounds, glassmorphism effects, and cyan accent colors
- **Responsive**: Mobile-friendly design that adapts to all screen sizes
- **Form Validation**: HTML5 input validation with visual feedback
- **Animated Transitions**: Smooth slide and fade animations with staggered timing
- **Icon Integration**: Font Awesome icons for visual enhancement

## Files

- `index.html` - Main HTML structure
- `style.css` - Styling and animations
- `style.js` - Toggle functionality between login/signup panels

## Usage

1. Open `index.html` in your browser
2. Click "Sign Up" to switch to registration form
3. Click "Sign In" to return to login form
4. Fill in required fields (inputs show validation on focus)

## Technologies

- HTML5
- CSS3 (animations, gradients, flexbox)
- JavaScript (DOM manipulation)
- Font Awesome 7.0.1

## Mobile Support

Optimized for devices with breakpoints at 768px and 480px
